# Asset attribution

The office tileset and map in this directory are vendored from
[`shahar061/the-office`](https://github.com/shahar061/the-office) (project code: ISC).

## Pixel art — LimeZu (IMPORTANT: non-commercial license)

The tilesets (`tilesets/*.png`) are LimeZu pixel-art assets, distributed under
the **LimeZu FREE VERSION license** (see `tilesets/LIMEZUASSETS-LICENSE.txt`):

- ✅ May be used **and edited** — **in non-commercial projects only**.
- ❌ May **not** be used or edited in commercial projects, and may not be resold.

This project (OfficeVibe) is a personal, non-commercial project, which is compatible
with that license. **If this project is ever commercialized, these tilesets must be replaced or a
paid LimeZu license obtained.**

(The on-screen Office cast is **not** LimeZu art — those sprites are drawn procedurally in
`scene/office/portraitArt.ts`. The original LimeZu character base sheets are no longer used and
have been removed.)

## Tiled map

`maps/office.tmj` is a Tiled JSON map from the same repo, built on the LimeZu tilesets above.
